# Landing AVEX Wanderson

Landing page dark premium para AVEX Ads, criada a partir do briefing em
`prompt-landing-avex.md` e da referência visual enviada.

## O que está incluído

- Hero dark com atmosfera quente, logo AVEX e notificações flutuantes.
- Headline, CTAs e seções conforme o briefing.
- Identidade AVEX com fundo preto, fonte Sora e acento laranja `#F15E22`.
- Assets locais da marca em `public/assets`.
- Testes de renderização para validar conteúdo, metadados e remoção do starter.

## Comandos

```bash
npm install
npm run dev
npm run build
npm test
```

## Deploy

O projeto está vinculado ao Sites em `.openai/hosting.json` e publicado pelo
pipeline do Sites.
